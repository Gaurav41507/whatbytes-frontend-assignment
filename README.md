# Whatbytes Frontend Assignment

A responsive e-commerce frontend built using Next.js, TypeScript, and Tailwind CSS.

## Live Demo

https://whatbytes-shopping-website-frontend.vercel.app/

## GitHub Repository

https://github.com/Gaurav41507/whatbytes-frontend-assignment

## Features

* Responsive Product Listing Page
* Product Detail Page with Dynamic Routing
* Search Functionality
* Category Filtering
* Price Range Filtering
* Featured Product Section
* Cart Badge UI
* Responsive Header, Sidebar, and Footer
* React Context API for State Management
* Built with Next.js App Router
* Tailwind CSS Styling

## Tech Stack

* Next.js
* TypeScript
* Tailwind CSS
* React Context API
* Lucide React
* React Icons

## Getting Started

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the project locally.
